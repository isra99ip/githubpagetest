Calculadora IMC con Machine Learning

Análisis técnico completo del sistema de aprendizaje automático para cálculo de Índice de Masa Corporal

¿Qué es este proyecto?

Este proyecto implementa un sistema de Machine Learning que aprende a calcular el IMC (Índice de Masa Corporal) utilizando regresión lineal múltiple. El sistema se entrena con datos que proporciona el usuario y mejora sus predicciones con cada nuevo dato.

Fórmula Tradicional: IMC = Peso (kg) ÷ Altura² (m)
Fórmula ML Aprendida: IMC = a × Altura + b × Peso + c

Métrica Descripción Interpretación
MAE (Error Absoluto Medio) Promedio de diferencias absolutas entre predicciones y valores reales < 1.0 = Excelente, < 2.0 = Bueno, < 3.0 = Regular
MSE (Error Cuadrático Medio) Promedio de diferencias al cuadrado, penaliza errores grandes Menor valor = Mejor modelo
R² (Coeficiente de Determinación) Proporción de variabilidad explicada por el modelo > 0.9 = Excelente, > 0.7 = Bueno
